# 🧮 Angular & CSS Learning Calculator

A simple, interactive web-based calculator built to master **Angular** fundamentals and modern **CSS** layouts.

## 🚀 Learning Goals
This project was created as a hands-on exercise to learn:
*   **Angular:** Component structure, data binding (`{{ }}`), event handling `(click)`, and basic services.
*   **CSS:** Flexbox/Grid layouts, responsive design (media queries), and custom styling (hover effects, transitions).
*   **Logic:** Implementing arithmetic operations and managing application state.

## ✨ Features
*   **Basic Arithmetic:** Addition, subtraction, multiplication, and division.
*   **Responsive UI:** Works on both desktop and mobile screens.
*   **Real-time Display:** Immediate visual feedback for every input.
*   **Error Handling:** Managing edge cases like division by zero.

## 🛠️ Built With
*   **Angular CLI** (v15+)
*   **TypeScript** for logic and type safety
*   **Pure CSS** (or Tailwind CSS / Bootstrap if used)

## 🔧 Installation & Setup
1.  **Clone the repo:**
    `git clone https://github.com`
2.  **Install dependencies:**
    `npm install`
3.  **Run the development server:**
    `ng serve`
4.  **Open in browser:**
    Navigate to `http://localhost:4200/`.
